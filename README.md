# ⚡ Pokemon Defense

Pokemon Defense adalah game browser berbasis HTML, CSS, dan JavaScript dengan konsep **tower defense bertema Pokémon**.

Player harus menaruh Pokémon sebagai tower untuk mempertahankan base dari serangan musuh yang datang secara bergelombang.

## 🎮 Features

- Pixel-art map
- Tower defense gameplay
- Pokémon attack system
- Enemy wave spawning
- HP bar & coin system
- Mobile friendly
- GitHub Pages support

## 🕹️ Gameplay

1. Mulai wave musuh
2. Musuh berjalan menuju base
3. Taruh Pokémon untuk menyerang musuh
4. Dapat coin dari enemy yang kalah
5. Bertahan selama mungkin

```bash

```

```bash

```
